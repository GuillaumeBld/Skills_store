# Meta-skills

Skills that help create, audit, manage, and improve other skills.

| Skill | Description |
|-------|-------------|
| skill-creator | Skill creation tooling and best practices |
| skill-library-manager | Manage skills lifecycle with GitHub integration |
| skill-quality-auditor | Audit skill quality and compliance |
| skill-test-generator | Generate tests for skills |
| skill-dependency-mapper | Map skill dependencies |
| skill-code-simplifier | Simplify skill code |
| skill-consistency-enforcer | Enforce consistency across skills |
| skill-example-validator | Validate skill examples |
| skill-documentation-generator | Generate skill documentation |
